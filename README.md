# Kata

## Installation

Create a Python virtual environment
```commandline
python -m venv venv
```

Activate the virtual environment:

On Linux / MacOS
```commandline
source venv/Scripts/activate
```

On Windows

```commandline
 .\venv\Scripts\activate
```

Install packages with pip
```commandline
pip install -r requirements.txt
```

## Run tests

Run test once
```commandline
pytest
```

Running tests continuously
```commandline
ptw
```
